Detailed Description

template<class E, Bool_t FIXED>
class panda::Container< E, FIXED >

Template class for all container implementations. Inherits from base_type::array_type or base_type::collection_type of the element class.

For a physics object class E that inherits from B, Panda containers inherit as Object <- ContainerBase <- ArrayBase <- Container<B, true>=Array <- Container<E, true>=Array<E> Object <- ContainerBase <- CollectionBase <- Container<B, false>=Collection <- Container<E, false>=Collection<E>
