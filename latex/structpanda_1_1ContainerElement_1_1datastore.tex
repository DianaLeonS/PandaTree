\subsection{Detailed Description}
Actual arrays and vectors written to the tree are members of datastore. For example, given a tree with branches particle.\-size/i particle.\-pt\mbox{[}particle.\-size\mbox{]}/\-F the \hyperlink{classpanda_1_1Particle}{Particle} datastore will have Float\-\_\-t pt\mbox{[}N\mbox{]}; which gets written to / read from the tree. The i-\/th \hyperlink{classpanda_1_1Particle}{Particle} instance on the other hand has Float\-\_\-t\& pt; which is a reference to datastore\-::pt\mbox{[}i\mbox{]}.
